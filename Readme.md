## 🚀 Быстрый старт

Выполните эти шаги, чтобы запустить проект локально на своем компьютере.

### 1. Клонирование репозитория
```bash
git clone https://github.com
cd имя-репозитория
```

### 2. Установка зависимостей
```bash
npm install
```

### 3. Запуск в режиме разработки
```bash
npm run dev
```
> ⚠️ **Важно:** Если при запуске возникает ошибка *Missing script: "dev"*, убедитесь, что вы находитесь в корневой папке с файлом `package.json` и данный скрипт прописан в конфигурации.
